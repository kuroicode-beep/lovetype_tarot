# Gemini CLI batch prompt for tarot generation

폴더 내의 flux.json 워크플로우를 사용해서 타로카드 78장을 배치 생성해줘.
ComfyUI 경로는 c:\comfyui 이다.

요구사항:
- 카드 비율은 일반적인 모바일 앱용 세로 카드 비율로 설정
- 해상도는 앱 카드 이미지로 쓰기 적당한 크기로 통일
- 전체 78장이 하나의 덱처럼 보이도록 동일한 화풍 유지
- 카드마다 달라지는 것은 카드 이름, 번호, 상징 요소만 반영
- 공통 스타일은 오리지널 웨이트 계열 구도감을 참고하되
  직접적인 기존 캐릭터/영화 장면 복제 없이
  서정적 hand-painted fantasy animation 분위기로 생성
- 카드명 텍스트는 이미지 안에 넣지 않음
- 테두리와 프레임은 통일감 있게 유지
- 파일명은 tarot_001 ~ tarot_078 형식으로 저장
- 실패한 카드가 있으면 목록을 따로 로그로 남김
- 먼저 5장 샘플 생성 후 결과 확인용 폴더에 저장하고
  그 다음 전체 78장을 생성
