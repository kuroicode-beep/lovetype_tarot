# LoveType Tarot — 상세 기능 정의

**앱 ID:** `lovetype-tarot` (클라이언트·서버 공통)  
**API Base:** `AppConstants.baseUrl` + `/api/v1`  
**문서 기준일:** 2026-03-27

---

## 1. 사용자·온보딩

| 기능 | 설명 | 구현 메모 |
|------|------|-----------|
| 인트로 | 앱 진입, 로그인/시작 진입 | `IntroScreen`, 배경 `AppAssets.bgIntro` |
| 회원가입 | Google 등 계정 연동 플로우 | `SignupScreen`, `AuthService` |
| 정보 입력 | 닉네임·성별·MBTI·생년월일 | `InfoInputScreen`, 생년월일로 소울카드 번호 계산(`calcSoulCard`) |
| 소울카드 공개 | 계산된 소울카드 설명·이미지 | `SoulCardScreen`, `soulCardImagePath` 등 |
| 메인 허브 | 일일/연애 리딩 진입·쿨타임·포인트 | `MainScreen`, `BottomBar` |

---

## 2. 리딩 플로우 (3카드 스프레드)

| 단계 | 화면 | 기능 |
|------|------|------|
| 키워드 | `KeywordScreen` | 주제별 태그 선택 |
| 카드 뽑기 | `CardDrawScreen` | 7장 중 3장 선택·순서 |
| 카드 확인 | `CardConfirmScreen` | 과거/현재/미래 라벨 확인 |
| 결과 | `ResultScreen` | 서버 AI 스토리 생성·TTS·공유·저장·포인트 차감 |

**데이터 전달:** `ReadingArgs` (topic, topicLabel, tags, cards).

---

## 3. AI 스토리 생성

- **엔드포인트:** `POST /api/v1/tarot`  
- **입력:** 프롬프트에 소울카드 설명, MBTI, 성별, 선택 카드·정역, 키워드 등 포함 (`ApiService.tarotReading`).
- **출력:** 본문 문자열을 화면에 표시.

---

## 4. 포인트·구독

| 항목 | 내용 |
|------|------|
| 잔액·구독 | `GET /payment/balance` — `PaymentService.refreshBalance` |
| 포인트 사용 | `POST /payment/use` — 연애 주제 등 차감 (`usePoints`) |
| 인앱 충전 | `IapService` — 스토어 구매 후 `POST /payment/charge` |
| 구독 상품 | `tarot_sub` 등 `productIds` 집합과 서버 상품 ID 정합 필요 |

**덱 표시:** 구독 여부(`isSubscribedNotifier`)가 웹툰 덱 선택에 사용됨 (`resolveTarotDeckVisual`).

---

## 5. 쿨타임

- **기준:** 서버가 단일 진실 원천(SSOT). 앱은 `GET /api/v1/tarot/cooltime` 응답의 `is_available`, `next_available_at`, `remaining_seconds` 계열 값을 파싱한다.
- **메인 화면:** `MainScreen` 진입 시 오늘/연애 주제 쿨타임을 서버에서 조회하고, 버튼 탭 시 해당 주제를 한 번 더 확인한 뒤 리딩 플로우로 진입한다.
- **카운트다운:** 서버가 준 `next_available_at`까지의 남은 시간을 화면에서만 1초 단위로 감소 표시한다. 로컬 저장소에 마지막 리딩 시각을 기록하지 않는다.
- **알림:** 결과 생성 후 서버 쿨타임을 다시 조회하고, 서버가 알려준 남은 시간 기준으로 쿨타임 해제 알림을 예약한다.

---

## 6. 히스토리·동기화

- 리딩 저장: `POST /tarot/history` (`saveTarotHistory`, `HistorySyncService`)
- 실패 시 로컬 큐(`keyHistoryPending`) 후 재전송 플로우 존재

---

## 7. 푸시·알림

- FCM 토큰 등록: `POST /push/register`
- 로컬 알림·스케줄: `NotificationService` + `timezone`
- 백그라운드 메시지: `FirebaseMessaging` 연동 파일 존재

---

## 8. 접근성·표시

- **고대비:** `AppState.toggleHighContrast` — 테마 전환 + 타로 덱 `low_vision_deck` 경로
- **TTS:** `TtsService` — 결과 화면에서 읽기
